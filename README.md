# London Crime Dashboard - individual assignment 1

An interactive Shiny dashboard exploring crime trends across London boroughs from 2008–2016.

## Live App
[View the deployed app here](https://ybaher-individual-assignment-532.share.connect.posit.cloud)

## Setup & Installation

### Requirements
- R 4.5.3 or higher

### Install packages
Open R and run:
```r
install.packages(c("shiny", "dplyr", "ggplot2", "bslib", "nanoparquet"))
```

### Run the app
```r
shiny::runApp()
```

## Data
Data sourced from [data.london.gov.uk](https://data.london.gov.uk) and available on 
[Kaggle](https://www.kaggle.com/datasets/LondonDataStore/london-crime/data).
